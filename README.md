Memory Match Game

Description:

The Memory Match Game is a fun and interactive game that challenges players to test and improve their memory skills. The player flips two cards at a time and tries to find matching pairs. If the cards match, they stay open; if not, they flip back. The game continues until all pairs are matched. This game helps improve concentration, memory, and logical thinking.

Technologies Used:

• HTML – Used to create the structure of the game (cards, buttons, layout)

• CSS – Used for styling, colors, animations, and responsive design

• JavaScript – Used to handle game logic such as card flipping, matching, scoring, and game completion

Instructions to Run the Game:

1. Create a project folder on your computer.

2. Inside the folder, add the following files:

• index.html

• style.css or gameStyle.css

• script.js or gameScript.css

3. Copy the game code into the correct files.

4. Open index.html using any web browser (Chrome, Edge, Firefox).

5. Click on the cards to start playing and match all pairs to win.
